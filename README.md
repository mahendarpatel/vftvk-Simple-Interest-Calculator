# github-final-project

## Simple Interest Calculator

This project is a Simple Interest Calculator created as part of the GitHub Final Project.

### Purpose
The purpose of this project is to demonstrate GitHub concepts such as repositories, branches, pull requests, and merges.

### How to Use
Run the simple-interest.sh script and enter:
- Principal
- Rate of interest
- Time period

The script calculates simple interest using the formula:
Simple Interest = (Principal × Rate × Time) / 100

### Example
Principal: 1000  
Rate: 5  
Time: 2  
Simple Interest = 100
